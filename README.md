# Cogni-Study
This project is aimed to increase the productivity of the student by tracking and analyzing their study hours.


CogniStudy is a modern student productivity and learning platform designed to help students manage studies, deadlines, notes, performance tracking, and AI-powered learning tools in one place.

🚀 Features
📅 Study Planner
⏰ Custom Deadline Timer
🤖 AI Tutor
📝 Notes Vault
📊 Performance Dashboard
👥 Community Section
⚙️ User Settings
🔐 Login & Signup Authentication
📱 Responsive UI Design
🛠️ Technologies Used
HTML5
CSS3
JavaScript


The goal of CogniStudy is to create an all-in-one academic companion for students that combines productivity, collaboration, and smart learning tools.
